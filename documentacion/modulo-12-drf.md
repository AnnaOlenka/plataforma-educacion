# Módulo 12: Introducción a Python - Django REST Framework

## Descripción General
Construye APIs REST potentes y flexibles con Django REST Framework. Aprende sobre Serializers, ViewSets, Autenticación y Permisos.

## Objetivos de Aprendizaje
- Comprender la arquitectura de REST y sus principios
- Crear Serializers para validar y transformar datos
- Construir ViewSets para operaciones CRUD automáticas
- Implementar autenticación con tokens
- Configurar permisos y autorizaciones
- Documentar APIs con Swagger/OpenAPI

## Estructura del Curso

### Módulo 1: REST API Basics
1. **Lección 1: Introducción a REST** (25 min)
   - ¿Qué es REST y sus principios?
   - Métodos HTTP: GET, POST, PUT, PATCH, DELETE
   - Códigos de estado HTTP

2. **Lección 2: Quiz - Django REST Framework** (20 min)
   - Evaluación de conceptos de API REST
   - Preguntas sobre Serializers, ViewSets y autenticación

## Temas Cubiertos
- Serializers y validación
- ViewSets y Routers
- APIView y Mixins
- Filtrado, búsqueda y paginación
- Autenticación: Token, JWT, OAuth2
- Permisos y controles de acceso
- Throttling y Rate Limiting
- Versionamiento de APIs
- Documentación automática: Swagger/OpenAPI
- Pruebas de APIs
