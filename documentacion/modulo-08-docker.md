# Módulo 08: Introducción a Docker

## Descripción General
Aprende los conceptos fundamentales de containerización y Docker, incluyendo imágenes, contenedores, Dockerfiles y Docker Compose.

## Objetivos de Aprendizaje
- Entender qué es un contenedor y cómo difiere de una máquina virtual
- Crear y ejecutar contenedores Docker
- Construir imágenes Docker personalizadas
- Usar Docker Compose para orquestar múltiples contenedores

## Estructura del Curso

### Módulo 1: Conceptos Básicos
1. **Lección 1: Introducción a Contenedores** (20 min)
   - ¿Qué es un contenedor?
   - Diferencias entre contenedores y máquinas virtuales
   - Ventajas de Docker

2. **Lección 2: Quiz - Conceptos Docker** (15 min)
   - Evaluación de conocimientos fundamentales
   - Preguntas de verdadero/falso sobre conceptos clave

## Temas Cubiertos
- Arquitectura de Docker
- Imágenes y capas
- Contenedores en ejecución
- Registros de Docker (Docker Hub)
- Volúmenes y redes
