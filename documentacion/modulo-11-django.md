# Módulo 11: Introducción a Python - Django

## Descripción General
Framework de Python de alto nivel para desarrollo web rápido y limpio. Aprende sobre Modelos, Vistas, Templates y el ORM de Django.

## Objetivos de Aprendizaje
- Comprender la arquitectura MVT de Django
- Crear modelos y migraciones
- Trabajar con el ORM de Django
- Construir vistas y templates
- Utilizar el admin de Django
- Implementar formularios y validación

## Estructura del Curso

### Módulo 1: Django Basics
1. **Lección 1: Introducción a Django** (25 min)
   - ¿Qué es Django y cuáles son sus características?
   - Patrón MVT (Modelo-Vista-Template)
   - Configuración inicial del proyecto

2. **Lección 2: Quiz - Django Concepts** (15 min)
   - Evaluación de conceptos fundamentales
   - Preguntas sobre modelos, vistas y templates

## Temas Cubiertos
- Estructura de proyectos Django
- Modelos y base de datos
- Vistas basadas en clases y funciones
- Templates y contexto
- Django ORM
- Migraciones de base de datos
- Admin automático de Django
- Formularios y validación
- Autenticación de usuarios
