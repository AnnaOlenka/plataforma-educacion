# Módulo 09: Introducción a Kubernetes

## Descripción General
Orquestación de contenedores con Kubernetes. Aprende sobre Pods, Servicios, Despliegues (Deployments) y la arquitectura general de Kubernetes.

## Objetivos de Aprendizaje
- Comprender la arquitectura de Kubernetes
- Trabajar con Pods y Deployments
- Exponer aplicaciones con Services
- Gestionar configuraciones y secretos
- Escalar aplicaciones horizontalmente

## Estructura del Curso

### Módulo 1: Fundamentos K8s
1. **Lección 1: Arquitectura de Kubernetes** (25 min)
   - Componentes del cluster (Master, Nodos)
   - Control Plane y Worker Nodes
   - Componentes clave: API Server, Scheduler, Controller Manager

2. **Lección 2: Quiz - Conceptos Kubernetes** (15 min)
   - Evaluación de conceptos arquitectónicos
   - Preguntas sobre componentes y responsabilidades

## Temas Cubiertos
- Cluster de Kubernetes
- Pods (unidades más pequeñas)
- Deployments (gestión de réplicas)
- Services (exposición de Pods)
- ConfigMaps y Secrets
- Volúmenes y almacenamiento
- Labels y Selectors

## Criterios de Aprobación
- Puntuación mínima: 70%
- Tiempo límite: 10 minutos

## Requisitos Previos
- Comprensión de Docker y contenedores
- Conocimiento básico de redes
