# Módulo 10: Introducción a PHP - Laravel

## Descripción General
Framework moderno de PHP para desarrollo web rápido y elegante. Aprende sobre Routing, Modelos, Migraciones y Eloquent ORM.

## Objetivos de Aprendizaje
- Entender la estructura MVC de Laravel
- Crear modelos y migraciones
- Trabajar con Eloquent ORM
- Construir rutas y controladores
- Utilizar el panel de administración de Laravel

## Estructura del Curso

### Módulo 1: Fundamentos Laravel
1. **Lección 1: Qué es Laravel** (20 min)
   - Introducción al framework
   - Ventajas y características principales
   - Instalación y configuración

2. **Lección 2: Quiz - Laravel Basics** (15 min)
   - Evaluación de conceptos fundamentales
   - Preguntas sobre MVC y componentes clave

## Temas Cubiertos
- Patrón de diseño MVC
- Routing y Controladores
- Modelos y Relaciones
- Migraciones de base de datos
- Eloquent ORM
- Validación de datos
- Panel de Administración (Artisan)
- Autenticación y Autorización

## Criterios de Aprobación
- Puntuación mínima: 70%
- Tiempo límite: 10 minutos

## Requisitos Previos
- Conocimiento básico de PHP
- Comprensión de bases de datos relacionales
- Familiaridad con HTML y HTTP
